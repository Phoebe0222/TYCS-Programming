### Setting up Scheme 
- install [Racket](https://racket-lang.org/download/) `brew install racket`
- start Racket  `$racket`
- install Simply Scheme package `(require (planet dyoo/simply-scheme:2:2))`
- load programs from files `(load "PATH_TO_SCM_FILE")`
